# React + TypeScript + Vite Admin Template

A lightweight yet engineering-friendly React template for admin/back-office projects, built for quick project kickoffs.

## Features

- Vite + React + TypeScript
- Router-friendly page structure
- Layout and error-boundary scaffolding
- State store and request utility placeholders
- Theme tokens and base styles
- ESLint/Biome/Lefthook rules and hooks
- Clean routing setup for multi-level nested pages

## Quick Start

```bash
pnpm install
pnpm dev
```

Open `http://localhost:5173`.

## Scripts

```bash
pnpm dev
pnpm build
pnpm preview
pnpm lint
```

## Configuration Notes

- Aliases and Vite config live in `vite.config.ts`.
- ESLint/Biome/Lefthook configs are included at the repo root.
